# Game-Events-Calendar
通过调用手游官网和Wiki，实时获取各大手游活动信息并整理到日历中，助您轻松规划游戏时间。

### DIY日历组件

- 具备日历基本的查看各个年份的日期功能

- 对于日程的显示：

  1. 缩略状态有日程的日期一个小点提示
  2. 完全展开可以看到一个时间条，上面有日程名称，长度对应起始与结束时间

- 响应式设计：pc端还能有侧边栏显示具体的活动内容 （次）

- 追加动效（次）

- UI设计参考+MIUI的日历
